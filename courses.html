Level 1: Getting started with tech

Computer Basics

Understand how your computer works, its components, and how to use them effectively.

Internet & Its Working

Learn how the internet connects everything, from websites to online services, and how it all functions together.

Web Development Basics

Get hands-on with creating your first website. You’ll learn the basics of HTML, CSS, and how to make your pages look good and work properly.

Programming Fundamentals

Start your coding journey by learning the fundamentals. You'll get familiar with basic programming concepts to build your first projects.

Linux & Scripting

Discover Linux, a powerful operating system used by professionals. Learn scripting to automate tasks and make your workflow more efficient.

Level 2: Expanding Your Knowledge

Frontend Development

Learn how to design beautiful, interactive websites using HTML, CSS, and JavaScript to create stunning front-end experiences.

Backend Development

Discover how websites work behind the scenes. Learn how to manage databases, servers, and APIs to deliver dynamic content.

Database Management

Learn how to organize, store, and retrieve data efficiently. Understand the basics of databases and how they power modern websites.

Cloud & Hosting

Explore cloud platforms and how to host websites. Learn how to deploy your projects on the web and make them accessible to everyone.

Containerization

Learn how to use Docker to package your apps and their dependencies. Ensure consistency across different environments for smooth deployment.

DevOps

Understand the collaboration between developers and operations. Learn automation tools to streamline building, testing, and deployment processes.

Git, GitHub & Actions

Master version control with Git, collaborate on projects using GitHub, and automate your workflow with GitHub Actions to improve productivity.

Level 3: Mastering the Craft

APIs

Learn how APIs allow different applications to communicate with each other. Build and integrate APIs to enhance your projects functionality.

Data Structures & Algorithms (DSA)

Master key data structures and algorithms to optimize code for performance and tackle complex problems with ease.

Design Patterns

Learn about design patterns, the best practices for solving common coding problems. Use them to write efficient, maintainable code.

Security

Learn the essential techniques to protect your websites and applications from security breaches. Safeguard sensitive data and prevent attacks.

Web Performance

Optimize your website’s speed and performance to ensure a seamless user experience. Improve load times and boost your site’s efficiency.

Accessibility

Make your website accessible to everyone, including users with disabilities. Follow accessibility best practices to ensure an inclusive design.

UI/UX Design

Learn the principles of user interface (UI) and user experience (UX) design to create intuitive, visually appealing websites and apps.

System Design

Learn how to design scalable, reliable systems. Understand architectural decisions that support growth, performance, and efficiency in large projects.
